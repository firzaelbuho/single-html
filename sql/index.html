Cara Pakai

Dashboard ini cocok untuk prototyping front-end aplikasi ekspedisi tanpa backend sungguhan.

Data awal ada di folder sql/data. Setelah browser membuka halaman, data akan disalin ke localStorage agar operasi create, update, dan delete bisa terasa nyata.

Jika ingin kembali ke data awal, hapus key ekspedisi-dashboard-db-v1 dari localStorage browser lalu refresh halaman.

Kalau file dibuka langsung via file://, beberapa browser memblokir fetch JSON. Script sudah saya beri fallback demo supaya halaman tetap jalan.
