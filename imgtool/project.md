IMG Tool

- start point index.html di folder ini (/imgtool)
- ada file css dan js script didalam folder assets
- ui utama pakai tailwind, baru kalau butuh lebih customize pakai css file
- ui minimmalis clean tapi nyaman
- responsive untuk berbagai perangkat


adalah sebuah web app berisi tool untuk gambar, user bisa memilih tool yg mau dipakai di sidebar kiri gitu.

tool :

1. IMG Resizer

- user bisa upload gambar, nanti bisa resize gambar sesuai preset yg ada (1:1, 4:3, 3:4, 16:9, 9:16) atau custom size resolusinya juga bebas
- bisa pilih format output (jpg, png, webp)
- jika ratio gambar tidak sama dengan ratio target, nanti gambar bisa di panning atau zoom in dan out biar sama dengan aspect ratio bebas



2. Icon Generator

- user upload gambar icon
- nanti bisa pilih ukuran output (16x16, 32x32, 48x48, 64x64, 128x128, 256x256, 512x512, dll custom yg penting 1:1, bisa multiple choices juga, nanti hasilnya jadi zip)
